## 📊 Descripción

**Descuubre Intelligence** es un dashboard profesional de Business Intelligence diseñado para visualizar y analizar datos turísticos de Quintana Roo. Proporciona insights clave sobre movilidad, zonas culturales, hospedaje y flujo de turistas.

### ✨ Características Principales

#### 🎯 **Dashboard Principal**
- KPIs clave del sector turístico
- Visualización de movilidad multimodal (Aéreo, Cruceros, Tren Maya)
- Distribución de origen de flujo turístico
- Métricas económicas y de ocupación hotelera

#### 🚆 **Movilidad 2026**
- Comparativa detallada: **Turismo Interno vs Externo**
- Análisis de llegadas aéreas (nacionales e internacionales)
- Métricas de uso del Tren Maya
- Desglose por modalidades de transporte

#### 🏛️ **Zonas Culturales**
- TOP 12 sitios arqueológicos de Quintana Roo
- Tulum, Chacchoben, Cobá, San Gervasio, Ichkabal y más
- Métricas de visitantes con tendencias de crecimiento
- Visualización de sitios con mayor afluencia

#### 🏨 **Red de Hospitalidad**
- Eficiencia operativa por destino turístico
- Categorización hotelera (5 estrellas, 4 estrellas, boutique)
- Tasas de ocupación por región
- Análisis de oferta hotelera activa

## 🚀 Instalación y Uso

### Requisitos Previos
- Node.js v18+ ([Descargar LTS](https://nodejs.org/))
- npm o yarn
- Git

### Instalación

```bash
# 1. Clonar el repositorio
git clone https://github.com/acampero-descuubre/descuubre-intelligence.git
cd descuubre-intelligence

# 2. Instalar dependencias
npm install

# 3. Ejecutar en modo desarrollo
npm run dev
```

### Build para Producción

```bash
# Generar build optimizado
npm run build

# Vista previa del build
npm run preview
```
